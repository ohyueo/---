# 海之心场地场馆|运动场馆运营订场小程序

官方网站 [www.ohyu.cn](https://www.ohyu.cn/index/app/)

#### 介绍
海之心场地场馆预约系统是一款针对运动场馆的运营管理系统，可用于，篮球场地预定、半场全场、羽毛球场地预定、乒乓球场地预定、网球足球预定等，前端由手机网页版+微信公众号+微信小程序组成，有订场、门票，赛事活动、商城、课程、会员卡等功能组成。
